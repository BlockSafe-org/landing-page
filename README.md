## BlockSafe - Landing Page Template with Next.js & Tailwind CSS

### 1\. Clone this Repository

```bash
git clone
```

### 2\. Navigate to the directory

```
cd [project directory]
```

### 3\. Install dependencies

```
yarn
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
